# SORM

## 简介✨✨✨✨✨✨

​	   sorm是一款基于go数据库操作包的orm框架，此项目并非完全的orm框架项目。而更像是一个开发日记，记录每次框架更新迭代的过程，版本号以v开发，例如 v0,v1,v2。

使用日记方式进行框架开发，旨在 😁

1. 加强对数据库系统了解，go语言与数据库交互     
2. 向大家记录开发框架全过程，供大家使用
3. 为大家提供pr机会，本人目前的技术水准应该还有待提高，欢迎大家指出🤠
4. 体验处理issue，与pr
5. 在实际的框架开发中，寻找
